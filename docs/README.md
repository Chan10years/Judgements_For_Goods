# 项目文档索引

本目录汇总项目说明、阶段计划、阶段总结和本次 GoodsSpider 改造总结。

## 主要文档

- `GFoodsSpider_SUMMARY.md`：本次 GoodsSpider 风格采购推荐系统改造总结。
- `QUICK_START_MODIFY_GUIDE.md`：快速了解项目、运行系统和上手改动的教学文档。
- `采购商品推荐系统改造总结与验收报告.docx`：正式 Word 总结报告。
- `PROJECT_CONTROL.md`：项目控制与执行记录。
- `PROJECT_ROADMAP_AND_COACHING.md`：项目路线与协作说明。
- `FLOW.md`：系统流程说明。
- `TASKS.md`：任务清单。

## 分类目录

- `plan/`：各阶段计划文档。
- `summary/`：各阶段总结、审计和验收文档。

## 未归入 docs 的文件

`data/` 和 `outputs/` 中的 Word 文件主要是模板、测试输入或运行输出，不属于项目说明文档，因此保留在原目录，避免影响程序运行和样例数据。
